# C++ Object-Oriented Programming Series

A comprehensive series of projects demonstrating core C++ concepts from fundamental memory management to advanced object-oriented design patterns.

## Project Overview

### cpp00 - Foundations

- **ex00**: Megaphone - String manipulation and I/O operations
- **ex01**: PhoneBook - Object design with data encapsulation and search functionality
- **ex02**: Account - Class design with state management and transaction logging

### cpp01 - Memory Management & Pointers

- **ex00**: Zombie - Dynamic allocation and proper resource deallocation
- **ex01**: Zombie Horde - Array allocation and heap memory management at scale
- **ex02**: Pointer vs Reference - Semantics and use-case differentiation
- **ex03**: Weapon & Humans - Object composition and reference semantics
- **ex04**: String Replacement - File I/O operations with stream manipulation
- **ex05**: Harl Filter - Function pointers for runtime polymorphism
- **ex06**: Harl Volume - State machine implementation with logging levels

### cpp02 - Operator Overloading & Arithmetic

- **ex00-ex02**: Fixed Point Numbers - Custom arithmetic operators and const correctness
- **ex03**: BSP Algorithm - Computational geometry with operator-based class design

### cpp03 - Inheritance Hierarchies

- **ex00**: ClapTrap - Base class design with member initialization
- **ex01**: ScavTrap - Single inheritance with method overriding
- **ex02**: FragTrap - Multiple inheritance management
- **ex03**: DiamondTrap - Solving the diamond problem with virtual inheritance

### cpp04 - Polymorphism & Abstraction

- **ex00**: Animal & Pets - Virtual functions and dynamic dispatch
- **ex01**: Brain - Deep copy semantics and resource management
- **ex02**: Abstract Classes - Pure virtual interfaces and polymorphic behavior
- **ex03**: Design Patterns - Abstract base classes for extensible architecture

## Building

Each exercise includes a `Makefile` for compilation:

```bash
cd <cpp0X>/<exYY> && make
./program_name
make clean
```
